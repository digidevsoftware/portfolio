# Portfolio

Simple static portfolio site intended for GitHub Pages.

## Publish steps

1. Create a GitHub repository, for example `portfolio`.
2. Push these files to that repository.
3. In GitHub, open **Settings -> Pages**.
4. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Your site will appear at:
   - `https://YOUR-USERNAME.github.io/portfolio/`

## Customize

Edit `index.html` to replace placeholders:
- name
- project descriptions
- contact links

Edit `styles.css` if you want a different look.
