# Nicholas Rohrmoser Portfolio

This repository contains the source for Nicholas Rohrmoser’s personal portfolio website.

## Live site

View the portfolio here:

- [https://digidevsoftware.github.io/portfolio/](https://digidevsoftware.github.io/portfolio/)

## About

This is a simple static portfolio site built for GitHub Pages. It highlights Nicholas Rohrmoser’s background, education, experience, technical skills, and projects as a graduate software engineer.

## Tech stack

- HTML
- CSS
- GitHub Pages

## Repository purpose

This repository is used to:
- maintain the live portfolio website
- update portfolio content over time
- publish CV and project information

## Updating the site

Typical content updates include:
- editing `index.html` for portfolio content
- editing `styles.css` for visual changes
- replacing `Nicholas_Rohrmoser_CV.pdf` when a newer CV is available

## Deployment

The site is deployed with GitHub Pages from the `main` branch.

Expected live URL:
- [https://digidevsoftware.github.io/portfolio/](https://digidevsoftware.github.io/portfolio/)
